# RestaurantSystem-Web

Nền tảng web quản lý nhà hàng (quản lý, báo cáo, đặt bàn/món online).  
Tech stack: ASP.NET MVC/WebForms, SignalR, SQL Server, JWT/OAuth2.

## Cấu trúc thư mục mẫu
- /Controllers
- /Models
- /Views
- /wwwroot
- /docs
- /DataSchema
